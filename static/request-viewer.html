LLM Gateway API Documentation

Interactive API documentation with live testing capabilities

chat Go to Chat Interface
POST /vendor-a/messages

Vendor A endpoint with canned responses. Returns response with detailed token usage and latency metrics. Has 10% error rate (500) and 10% slow response rate. Supports optional invoice lookup tool.

Request Body
Parameter Type Description
prompt string The message to send to the model
system_prompt string Optional system instruction for the model
Response
{
  "outputText": "Model generated response",
  "tokensIn": 15,
  "tokensOut": 25,
  "latencyMS": 1234
}
POST /vendor-b/messages

Vendor B endpoint using DistilGPT-2 model. Returns OpenAI-style response format with choices array. Has 10% rate limit error (429).

Request Body
Parameter Type Description
prompt string The message to send to the model
system_prompt string Optional system instruction for the model
Response
{
  "choices": [{
    "message": {
      "content": "Model generated response"
    }
  }],
  "usage": {
    "input_tokens": 15,
    "output_tokens": 25
  }
}
POST /vendor-e/messages

Vendor E is an echo endpoint that simply returns your input. Useful for testing and validation.

Request Body
Parameter Type Description
prompt string The message to echo back
Response
{
  "response": "You entered {your prompt}"
}
POST /vendor-o/messages

Vendor O is a passthrough to OpenAI's API. Returns standard OpenAI response format. Requires OPENAI_API_KEY environment variable to be set.

Request Body
Parameter Type Description
prompt string The message to send to OpenAI
system_prompt string Optional system instruction (default: "You are a helpful assistant")
model string OpenAI model to use (default: "gpt-3.5-turbo")
tools array Optional list of tools/functions the model can call
tool_choice string Optional tool choice: "auto", "none", or specific tool name
Response
{
  "id": "chatcmpl-...",
  "object": "chat.completion",
  "created": 1234567890,
  "model": "gpt-3.5-turbo",
  "choices": [{
    "index": 0,
    "message": {
      "role": "assistant",
      "content": "Response from OpenAI"
    },
    "finish_reason": "stop"
  }],
  "usage": {
    "prompt_tokens": 15,
    "completion_tokens": 25,
    "total_tokens": 40
  }
}
